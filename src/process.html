Arbetsprocess

Beskrivning av skapandet av denna sida med automatisering och publicering i fokus

  1. Jag började med att installera ett parcel-paket i NodeJs och packeterade det i en JSON-fil i Visual Studio Codes
  2. I paketet angav jag två script för att kunna starta och uppdatera (run / build) och la även till en bakåtkompabilitet för att webbplatsen ska kunna fungera i flertalet webbläsare samt testade en ominstallering (npm install) för att kunna ladda ner paketen igen på ev. ny dator istället för att flytta dem
  3. Jag la därefter i mina html-filer, bilder, JS och CSS och arbetade i dessa för att skapa webbplatsens design och struktur
  4. Efter det skapade jag ett publikt repo på Github och pushade mitt arbete från Visual Studio Codes dit och fortsatte därefter att arbeta på innehållet och skickade regelbundet commits till mitt repo vid ändringar och tillägg
  5. Bilderna på undersidan "bilder" har lagts i ett picture-element med en angiven storlek och därefter optimerats med hjälp av parcel och omvandlats från det tillagda formatet JPG till det angivna WEBP
